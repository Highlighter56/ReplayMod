# ReplayMod

A puzzle game based on recording player actions and replaying them to solve levels.

## About
This project explores a record-and-replay mechanic where past actions are replayed to interact with the environment and complete puzzles.

## Built With
- Godot Engine

## How to Run
### Option 1
- Check out my [Itch.io page](https://highlighter56.itch.io/project-replay)
### Option 2
- Open the project in Godot
- Press **Play** to start the game
